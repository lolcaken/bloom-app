Bloom Watch App

NASA Space Apps Challenge 2025

Moath & Hafsa 8B

Blooming Flower Data Logger

Project Overview

The "Bloom Watch App" is an interactive tool designed to track and manage data on flower blooming events across different continents. It allows users to report new sightings and query existing records, making it a valuable resource for environmental monitoring, research, and education.

By collecting data on the location, size, and timing of blooms, the app creates a centralized database that can be used to identify patterns, study ecological changes, and inform conservation efforts. Its simple command-line interface makes it accessible for anyone to contribute to or explore the global bloom record.

The Challenge

Develop an intuitive system for tracking and analyzing flower blooming data globally, enabling both contributions from citizen scientists and easy access to information for researchers and enthusiasts.

Key Features

  • Data Collection: Users can report new flower blooming instances by specifying the continent, number of hectares, month, and year.
  • Interactive Querying: Retrieve stored bloom data by filtering through month, year, and continent criteria.
  • Centralized Storage: All reported data is saved and managed within the application for future reference and analysis.
  • User-Friendly Interface: A straightforward text-based interface guides users through the process of reporting and searching for data.

Key Considerations

  • Ensuring data integrity and ease of entry during the reporting phase.
  • Making stored information easily searchable and understandable.
  • Providing clear feedback to users after each operation (e.g., confirming data entry or reporting no results).
  • Designing the system to be scalable for a growing dataset.
